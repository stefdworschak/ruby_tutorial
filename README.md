# ruby_tutorial

## Installing Rails

### Make sure Ruby and Node.js are installed

`ruby -v`
`node -v`

### Configure Ruby Gem

`gem update --system`

### Install Rails

`gem install rails -v 5.2.0`

### Update bundler

`gem i bundler -v 1.17.3`
`bundle i`

## Allow hosts

Add the following line to your `application.rb` file

`config.hosts << "[HOSTNAME]"`
